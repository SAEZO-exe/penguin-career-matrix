# Penguin Coder | Career Intelligence Platform

A high-performance career intelligence platform built for students and engineering professionals. **Penguin Coder** helps users deconstruct industry career fields, access structured learning roadmaps, and calculate real-time career ROI metrics localized for the Indian market.

## 🚀 Key Features

*   **Heuristic Career Optimizer**: Dynamically maps market data (₹ LPA) for any job role using a fuzzy-matching search engine.
*   **AI Roadmap Generator**: Powered by Gemini AI, it instantly generates a custom 5-stage syllabus for any profession.
*   **ROI Intelligence Matrix**: Real-time side-by-side comparison of salary returns versus preparation effort.
*   **Smart Chat Oracle**: Integrated AI counselor for personalized market and academic advice.
*   **Performance-Driven UI**: Built with a "Liquid Glass" theme optimized for speed and readability.

## 🛠️ Built With

*   **Frontend**: HTML5, CSS3, Vanilla JavaScript
*   **AI Integration**: Google Gemini API (via Google AI Studio)
*   **Data Logic**: Custom heuristic and string-distance algorithms

## 📦 How to Run

1. Clone this repository to your local machine.
2. Open `index.html` in your browser.
3. For AI features: Generate a free API Key from [Google AI Studio](https://aistudio.google.com/) and paste it into the "Vault" field on the dashboard.

## 📈 Platform Goals

*   **Empowerment**: Providing data-driven clarity for students navigating the Indian job market.
*   **Efficiency**: Reducing search time by synthesizing professional roadmaps on-demand.
*   **Transparency**: Calculating actual ROI indices for complex career transitions.
